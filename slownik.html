Krótki wstęp: Wstęp

HTML
HyperText Markup Language — język znaczników używany do tworzenia stron internetowych. Przykład: <p>Tekst</p>
CSS
Cascading Style Sheets — stylowanie elementów HTML, np. color, font-size.
tag
Znacznik — składnik <h1>, <p> definiujący strukturę dokumentu.
atrybut
Atrybut — dodatkowa informacja dla znacznika, np. href w <a>.
element
Element — kompletny komponent, np. <a>Link</a>, składa się ze znacznika otwierającego i zamykającego.
DOM
Document Object Model — reprezentacja dokumentu jako obiektów w pamięci przeglądarki.

Znaczniki formatowania

  • <strong> — pogrubia tekst (wyraża wagę).
  • <em> — kursywa (nacisk).
  • <u> — podkreślenie.
  • <s> — przekreślenie.
  • <mark> — wyróżnienie (tło).
  • <sup> — indeks górny.
  • <sub> — indeks dolny.
  • <code> — fragment kodu monospaced.
  • <blockquote>
    — zacytowana treść.
